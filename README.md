# Banco Ejemplo — Información de la página web

Descripción
-----------
Este repositorio contiene una descripción básica con la información pública tomada de la página web oficial de `Banco Ejemplo`.

Sitio web
---------
https://www.bancoejemplo.com

Servicios principales
--------------------
- Banca en línea: consulta de saldos, movimientos y transferencias.
- Banca móvil: aplicaciones iOS y Android para operaciones rápidas.
- Productos financieros: cuentas de ahorro, cuentas corrientes, préstamos y tarjetas de crédito.
- Atención al cliente: chat en línea, líneas telefónicas y sucursales.

Horario de atención en línea
---------------------------
- Atención 24/7 para servicios básicos en la plataforma digital.
- Atención telefónica: Lunes a Viernes 8:30–18:00 (hora local).

Contacto y soporte
------------------
- Teléfono: +1 800 000 000
- Correo de soporte: soporte@bancoejemplo.com
- Chat: disponible en https://www.bancoejemplo.com/chat

Seguridad y privacidad
----------------------
- Autenticación: acceso mediante usuario y contraseña; soporte para autenticación de dos factores (2FA).
- Cifrado: todas las comunicaciones web usan HTTPS/TLS.
- Privacidad: el banco publica su política de privacidad en https://www.bancoejemplo.com/privacidad

Enlaces útiles
